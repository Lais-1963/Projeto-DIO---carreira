Aqui vai um panorama bem claro e direto:

O que faz um Desenvolvedor Web Full Stack?
É o profissional capaz de trabalhar tanto no front-end (parte visual) quanto no back-end (lógica, servidores e bancos de dados) de aplicações web.
Você cria sites, sistemas e aplicações completas, desde a interface até o funcionamento interno.

O que você aprende normalmente?
Frontend (o que o usuário vê)
* HTML
* CSS
* JavaScript
* Frameworks: React, Vue ou Angular
Backend (o que roda nos bastidores)
* Node.js, Python ou outra linguagem
* APIs
* Bancos de dados (SQL ou NoSQL)
* Autenticação, segurança básica
* Arquitetura de aplicações
Ferramentas gerais
* Git e GitHub
* Deploy (colocar o projeto no ar)
* Versionamento
* Métodos ágeis (Scrum, Kanban)

Perfil ideal
Combina muito com quem:
* gosta de resolver problemas
* curte ver resultado rápido
* quer trabalhar com código
* se interessa por criar produtos
* deseja uma carreira com muita demanda para iniciantes
Tudo isso bate com o que você respondeu.

Crescimento na carreira
Há vários caminhos possíveis depois de entrar:
* Especialista em front-end
* Especialista em back-end
* Engenheiro Full Stack avançado
* Engenheiro de Software
* Arquiteto de Software
* Fazer transição natural para IA aplicada ao desenvolvimento, se quiser

Mercado
O mercado de desenvolvedores web segue forte e costuma absorver muitos iniciantes, especialmente quem tem projetos práticos.
(Claro, sempre varia por região e experiência.)
