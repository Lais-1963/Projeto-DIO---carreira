# Projeto-DIO---carreira
Projeto DIO - Planejamento de carreira em tecnologia por agentes Copilot -
Foram utilizados os dois agentes, um entrevistador que definiu áreas de maior aderência e outro, a partir dados obtidos, que processou e apresentou um planejamento de estudos para capacitação (no caso estendi a duração para 180 dias).
